### Installation:
* Clone this repository to your machine
* Modify or replace ```input.txt``` as you see fit
* Compile with ```gcc -Wall main.c readfile.c -o workerDB```
* Run with ```./workerDB input.txt``` or ```.workerDB <your_file_name>.txt```
